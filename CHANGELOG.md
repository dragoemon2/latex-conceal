# Change Log

All notable changes to the "latex-conceal" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [Unreleased]

- No unreleased changes yet.

## [0.2.0] - 2026-03-25

### Added

- Automatically load replacement mappings from `\newcommand`, `\renewcommand`, and `\DeclareMathOperator` definitions.
- Added changelog. Note that previous log until this version is not accurate since it was written retrospectively based on commit history.

## [0.1.2] - 2026-03-24

### Added
- Added configurable replacement glyph color with `latex-conceal.replacementColor`.

## [0.1.2] - 2026-03-23

### Changed
- Updated extension icon to 1024x1024.

## [0.1.1] - 2026-03-23

### Added
- Added extension icon

## [0.1.0] - 2026-03-23

### Changed
- Improved conceal rendering performance significantly.
- Simplified internal implementation and removed unnecessary debug/debounce logic.
